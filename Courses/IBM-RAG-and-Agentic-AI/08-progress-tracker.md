## How to Use This Tracker

1. Resuming after a break → check this file first, then jump to the
   relevant track doc for the course you're mid-way through.
2. Finishing a lab → check it off in the relevant track doc (not here —
   this file stays at course-level granularity).
3. Finishing a course → update Status/Completed date/Hours here, then
   fill in that course's "Key takeaways" in the track doc.
4. Learning something genuinely new and reusable → merge into the
   matching canonical doc immediately, don't batch it for later (batching
   is how notes piles happen).

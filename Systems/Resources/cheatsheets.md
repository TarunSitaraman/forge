# Cheatsheets

*Fast external lookup references — for anything Forge's own
`Systems/Docs/` cheatsheet sections don't yet cover in enough depth.
Check `Systems/Docs/<technology>.md` first; only add an external
cheatsheet here for genuine gaps.*

---

## Admission criteria

Add an external cheatsheet only when:
- It covers something not already in a `Systems/Docs/` cheatsheet section
  (if it overlaps, the fix is to improve the `Systems/Docs/` entry, not
  duplicate content here).
- It's from a source likely to stay maintained/accurate (official docs'
  own quick-reference pages are usually the safest bet).

## How to use this page

If you find yourself reaching for the same external cheatsheet
repeatedly, that's a signal to fold its content into the relevant
`Systems/Docs/<technology>.md` cheatsheet section instead — Forge's own
reference should absorb genuinely recurring lookups over time.

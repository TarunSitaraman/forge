# Courses

*Structured courses worth the real time investment they require —
courses are the highest-time-cost resource in this module, so the bar
for adding one should be correspondingly higher than a blog post or
video.*

---

## Admission criteria

Add a course only after actually completing it (or a substantial portion)
and can state:
- What it taught that self-directed reading/tutorials wouldn't have
  covered as well — usually structure, sequencing, or hands-on exercises
  with feedback.
- Who it's actually for (beginner/intermediate/advanced) — a course
  recommendation without a level is a common source of wasted time.
- Whether it's still current (courses on fast-moving tools go stale
  faster than books or docs).

## How to use this page

Reach for a course when you want structured, sequenced learning with
exercises — for a fast lookup or quick orientation, official docs or a
`Systems/Docs/` entry will usually be faster.

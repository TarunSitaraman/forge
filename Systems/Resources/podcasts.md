# Podcasts

*Podcasts worth the commute/gym time — the bar is the same as
newsletters: you actually finish episodes, not let them pile up unplayed.*

---

## Admission criteria

Add a podcast only after listening to several episodes and confirming
you'd recommend a specific episode to a colleague. Note which episodes
were exceptional, not just that the show overall is decent.

## Maintenance

If a show's queue has been building up unplayed for a while, that's
signal to remove it rather than let this page reflect aspiration instead
of actual listening habits.

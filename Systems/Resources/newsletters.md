# Newsletters

*Newsletters worth the inbox space — worth it means you actually read
it when it arrives, not archive it unread. Prune ruthlessly; an unread
newsletter subscription is worse than no subscription.*

---

## Admission criteria

Add a newsletter only after receiving several issues and confirming you
actually read them. State specifically what it surfaces that you
wouldn't otherwise catch (a specific beat/domain it covers well).

## Maintenance

Review this list every few months against your actual inbox behavior —
if you've been archiving an entry unread for the last several issues,
unsubscribe and remove the row. This page should reflect what you
actually read, not what you once thought you'd read.

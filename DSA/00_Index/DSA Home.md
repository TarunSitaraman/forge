## How To Use This Section
1. Start with the problem signal and open [[Pattern Index]].
2. Follow links to the relevant algorithm, data structure, and template.
3. Solve or review the problem in [[Problem Index]].
4. Record recurring failure modes in the mistake tracker.
5. Use cheat sheets only for final recall, not canonical depth.

## Canonical Rule
If a page explains a concept in depth, other pages should summarize it briefly and link back. This keeps Forge durable and prevents conflicting explanations from spreading through the graph.

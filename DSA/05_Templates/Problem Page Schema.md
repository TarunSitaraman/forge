# Problem Page Schema

## Metadata
Use Forge frontmatter with platform, difficulty, patterns, algorithms, status, and review dates.

## Problem Statement
Summarize the prompt in your own words and link the external source when allowed.

## Pattern
Link canonical pattern pages such as [[Binary Search]] or [[Sliding Window]].

## Python Solution
Include clean, tested Python code.

## Reasoning
Explain the invariant, state transition, or decision rule.

## Complexity
State time and space complexity with the reason.

## Lessons
Link reusable insights to [[Lesson Template]].

## Mistakes
Link mistakes such as [[Off-by-One]] or [[Wrong Mid Calculation]].

## Related Problems
List similar problems and what changes.

## Cross-links
Link templates, algorithms, data structures, interview notes, and cheat sheets.

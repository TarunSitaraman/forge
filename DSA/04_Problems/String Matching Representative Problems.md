# String Matching Representative Problems

Core variants exercising String Matching algorithms.

## Variant 1: Pattern Matching
- KMP (Knuth-Morris-Pratt) algorithm
- Boyer-Moore algorithm
- Rabin-Karp (rolling hash) method

## Variant 2: Substring Problems
- Find all occurrences of pattern
- Shortest substring containing pattern
- Pattern appears at every position

## Variant 3: Multiple Pattern Matching
- Aho-Corasick algorithm for k patterns
- Finding any of k patterns efficiently
- Optimal multi-pattern search

## Variant 4: Fuzzy Matching
- Edit distance (Levenshtein)
- Wildcard matching patterns
- Regular expression matching

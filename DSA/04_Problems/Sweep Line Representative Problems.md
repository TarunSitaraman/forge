# Sweep Line Representative Problems

Core variants exercising Sweep Line technique for interval/coordinate problems.

## Variant 1: Event Processing
- Process events in sorted order (start/end)
- Track active elements/intervals
- Calculate total coverage

## Variant 2: Interval Covering Problems
- Skyline problem (maximum height)
- Merge intervals variation
- Visible points from origin

## Variant 3: Coordinate Compression
- Points in rectangle queries
- Circle coverage problems
- Area calculation with overlaps

## Variant 4: Optimization Problems
- Minimum event rooms needed
- Minimum meetings for scheduling
- Optimal event ordering

## Variant 5: Temporal Processing
- Time-based event handling
- Chronological sorting
- Duration tracking

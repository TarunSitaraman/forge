# Monotonic Queue Representative Problems

Core variants exercising Monotonic Queue for sliding window optimization.

## Variant 1: Sliding Window Maximum/Minimum
- Maximum in every window of size k
- Minimum in every window
- Custom aggregation function

## Variant 2: Deque-Based Optimization
- Maintaining decreasing order in deque
- O(1) amortized per window
- Front access for maximum efficiently

## Variant 3: Tricky Variations
- Multiple queries on same array
- Different window sizes
- Aggregate statistics (sum, product)

## Variant 4: Complex Windows
- Shrinking/expanding windows
- Adaptive window sizing
- Constraint-based window

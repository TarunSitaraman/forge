# Memoization Representative Problems

Core variants exercising Top-Down DP with Memoization.

## Variant 1: Fibonacci & Simple Recursion
- Fib(n) with memoization transforms O(2^n) to O(n)
- Cache intermediate results
- Single-variable state

## Variant 2: Tree DP Problems
- Tree pruning with memoization
- Max/min path in tree with constraints
- Subtree optimization queries

## Variant 3: Combinatorial Problems  
- Combinations with memoization
- Partitions counting
- Distributing items with constraints

## Variant 4: Game Theory
- Winning/losing positions with memo
- Optimal play scenarios
- Minimax with memoization

## Variant 5: String & Sequence
- Word break with memoization
- Decode ways
- Maximum subarray optimization

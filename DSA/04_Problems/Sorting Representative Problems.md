# Sorting Representative Problems

Core variants exercising Sorting algorithms and applications.

## Variant 1: Comparison-Based Sorts
- Merge sort O(n log n) guaranteed
- Quick sort O(n log n) average, O(n²) worst
- Heap sort O(n log n) guaranteed
- Selection sort O(n²) simple

## Variant 2: Non-Comparison Sorts
- Counting sort O(n+k)
- Radix sort O(d*(n+k))
- Bucket sort O(n+k)

## Variant 3: Sorting Applications
- Sort by custom comparator (objects)
- Largest number by rearrangement
- Reorder by specific condition

## Variant 4: Partial Sorting
- Kth smallest/largest element
- Merge k sorted arrays
- Top k elements (heap)

## Variant 5: Stability & Properties
- Stable sort preservation
- In-place sorting
- External sorting for massive data

# Trie Representative Problems

Core variants exercising Trie (Prefix Tree) data structure.

## Variant 1: Autocomplete & Prefix Search
- Word search by prefix
- Autocomplete suggestions
- Prefix matching efficiency

## Variant 2: Word Dictionary
- Add/search words in dictionary
- WordDictionary class design
- Wildcard matching (* and .)

## Variant 3: Longest Prefix
- Longest common prefix among words
- Maximum word prefix in dictionary
- Shared prefix finding

## Variant 4: Map Sum Pairs
- Sum of map values with prefix key
- Prefix-based aggregation
- Weighted trie operations

## Variant 5: Stream Processing
- Unique emails with domain filtering
- Stream processing with trie storage
- Efficient lookup and filtering

## Variant 6: Word Ladder & Variants
- Word transformation paths
- Connected words via character swaps
- Graph on trie structure

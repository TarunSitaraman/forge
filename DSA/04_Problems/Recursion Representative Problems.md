# Recursion Representative Problems

Core variants exercising Recursion and recursive thinking.

## Variant 1: Simple Recursion
- Factorial, fibonacci
- Sum of array
- Linear search in array

## Variant 2: Tree Recursion
- Tree traversal (preorder, inorder, postorder)
- Tree path finding
- Tree construction from sequences

## Variant 3: Backtracking Recursion
- Permutations generation
- Combinations generation
- Subset generation (power set)

## Variant 4: Divide-Conquer Recursion
- Merge sort via recursion
- Binary search recursively
- Closest pair problem

## Variant 5: Mutual Recursion
- Oddness/evenness checking via mutual calls
- Grammar parsing
- Co-recursive definitions

## Variant 6: Tail Recursion
- Tail call optimization potential
- Accumulator pattern
- Linear recursion variants

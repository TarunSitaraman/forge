
## 📞 If You Hit Issues

| Issue | Solution |
|-------|----------|
| Description doesn't show | Refresh the page; sometimes GitHub takes a moment |
| Topics won't save | Make sure you press Enter after each topic |
| Links broken in README | Check the full path of files you're linking to |
| Emoji shows as question mark | Different Markdown renderers handle emoji differently; this is fine |
| README looks different than expected | GitHub's Markdown rendering varies; check on mobile too |

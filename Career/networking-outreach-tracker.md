# Networking Outreach Tracker

*Guidance: networking that isn't tracked turns into a pile of
unanswered messages you forget to follow up on. This tracker exists to
make follow-up and reciprocity visible — not to gamify a relationship
into a numbers game.*

## Follow-up rules

- If no response after 1 week: one polite, brief follow-up — do not
  chase further than this uninvited.
- If they respond and help in any way (advice, a referral, an intro):
  log it, and set a reminder to update them on the outcome later — this
  is what turns a single interaction into an actual relationship.
- Never let "Status: Stalled" sit unreviewed for more than a month —
  either follow up once more or mark it closed.

## Best practices
- Log the outreach the same day you send it — a tracker updated
  retroactively from memory misses half the follow-ups that matter.
- Review this monthly: which warm contacts haven't been followed up
  with, and which relationships are worth deepening before you need
  something from them.
